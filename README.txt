# Hotel Games CMS Starter

This starter kit gives you:

- admin.html — create posts
- news.html — displays News posts
- media.html — displays Media posts
- community.html — displays Community posts
- cms.css — Hotel Games red/black styling
- cms-local-demo.js — demo storage using browser localStorage

## How to test now

1. Put these files in the same folder as hotelgamessticker.png.
2. Open admin.html.
3. Create a post.
4. Open news.html, media.html, or community.html depending on the category.

## Next step: Firebase

To make posts save online instead of only in your browser, create a Firebase project and enable:

- Authentication
- Firestore Database
- Storage

Then replace cms-local-demo.js with a Firebase version using your Firebase config.
